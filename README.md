### Algorithms implemented 
* RGB to Gray
* Contrast Limited Adaptive Histogram Equalization
* Bilateral Filtering
* Gaussia Blurring
* Mean Shift Segmentation
* Median Filtier
* Sobel Filter
* Stereo Matching
